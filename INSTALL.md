# Установка CraftCMS

## Что нужно

- VPS с Ubuntu 20.04 / 22.04 / 24.04 или Debian 11 / 12
- Минимум 1 GB RAM, 10 GB диск
- Открытые порты 80 и 443
- **Токен доступа** — ты получил его при покупке

Больше ничего не нужно. Скрипт сам установит Java, Node.js и всё остальное.

---

## Установка (3 команды)

Подключись к серверу через SSH и выполни по очереди:

```bash
# 1. Скачай установщик
wget https://raw.githubusercontent.com/struggler075/CMSMinecraft/main/install.sh

# 2. Запусти его
sudo bash install.sh
```

Скрипт спросит:
- **Токен** — вставь тот что получил при покупке
- **Домен или IP** — адрес твоего сервера
- **SSL** — нажми y если у тебя есть домен (делает https://)

Затем подожди 5–15 минут. Скрипт сам всё установит и запустит.

---

## Управление

```bash
# Статус — работает ли сайт
systemctl status craftcms

# Перезапуск
systemctl restart craftcms

# Логи в реальном времени (Ctrl+C чтобы выйти)
journalctl -u craftcms -f

# Обновление до новой версии
sudo bash update.sh
```

---

## Если что-то не работает

Выполни команду и пришли нам результат — мы разберёмся:

```bash
journalctl -u craftcms -n 50 --no-pager
```
